Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via tables and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings including a heading (level 1–6)
<p></p> Paragraph It is used to align text into paragraphs
table table It is used to include table
tr table row It is used to include table row
th table heading it is used to include table heading
td table data it is used to include table data cell
header header it is used for holding tags
div division it is a block element used for dividing html into logical blocks for styling
span span it is an inline element used for dividing html into logical blocks for styling
ol Ordered list for creating ordered list of a given items
ul Unordered list for creating unordered list of a given item
li list for creating list either ordered or unordered.

Inline Elements

Tag Name Purpose Example Result
em emphasized Make emphasized text that is italics <em>technical sophistication</em> technical sophistication
<strong></strong> Strong For emphasized text that is bolden <strong>technical sophistication</strong> Testing for strong tag
<a></> Anchor tag For creating links <a>click here</> click here
<img> Image tag for rendering images <img scr="images/storm_trooper_tagged.jpg">
